The thesis

Kubernetes was right for 2014. It is not right for 2026.

Stable eBPF APIs, kTLS offload, production-ready Rust systems libraries, and embeddable WASM only matured in the last two years. Overdrive is the platform that becomes possible when all of them exist at the same time.

01 / Own your primitives

No etcd. No Envoy. No SPIRE. No CNI.

Every critical subsystem is built into the platform or is a standard Rust library. External process dependencies are operational liabilities — they get cut.

02 / eBPF is the dataplane

The kernel does the work the proxy used to do.

Service routing, network policy, load balancing, mTLS, and telemetry happen at line rate in the kernel via aya-rs. No userspace proxies in the data path.

03 / Security is structural

mTLS between workloads cannot be turned off.

Every connection is wrapped in kTLS with a SPIFFE identity. Policy is enforced by BPF LSM in-kernel. A compromised workload hits the same walls as a well-behaved one.

Job spec

Declare what. The platform handles how.

A single TOML job spec composes drivers, sidecars, security profiles, and ingress. The Image Factory turns schematics into reproducible, content-addressed node images.

job.toml 
# AI agent with structural credential isolation
# and prompt-injection scanning.

[job]
name   = "ai-research-agent"
driver = "wasm"

[[job.sidecars]]
name    = "credential-proxy"
module  = "builtin:credential-proxy"
hooks   = ["egress"]

  [job.sidecars.config]
  allowed_domains = ["api.anthropic.com"]
  credentials     = { ANTHROPIC_KEY = { secret = "prod" } }

[[job.sidecars]]
name   = "content-inspector"
module = "builtin:content-inspector"
hooks  = ["ingress"]

[job.security]
no_raw_sockets          = true
no_privilege_escalation = true
egress.mode             = "intercepted"
~/.overdrive

Native SRE agent

An LLM that reasons about your cluster — safely.

Every eBPF event carries cryptographic SPIFFE identity. Correlation isn't a label-matching heuristic — it's a SQL join. The investigation agent proposes typed actions through a graduated approval gate. It cannot do anything the platform doesn't already understand.

  • Investigations as a resource. Lifecycle, budget, transcript, compressed into incident memory on conclusion.
  • Typed remediation. Tier 0 reads auto-execute, Tier 2 writes require human ratification.
  • Hypothesis verification. The agent can attach signed BPF probes for one investigation turn.
  • Deterministic replay. Investigation transcripts re-run in CI under the DST harness.

vs Kubernetes

Coherent by construction, not by configuration.

When the dataplane, identity model, telemetry pipeline, and service mesh all emerge from the same kernel primitive with the same workload identity attached, the platform becomes coherent in a way that bolted-on approaches cannot match.
